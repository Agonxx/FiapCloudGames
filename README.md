# FIAP Cloud Games — Tech Challenge Fase 1

API REST desenvolvida em **.NET 8** para o projeto **FIAP Cloud Games**, contendo o MVP inicial com cadastro de usuários, autenticação com JWT e estrutura organizada por camadas seguindo princípios de DDD.

---

## ✅ Tecnologias

- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
- Swagger
